| 状态             | 描述                                                                                                      |
| -------------- | ------------------------------------------------------------------------------------------------------- |
| **已验证**        | 提交已签名，签名已成功验证，并且提交者是启用警戒模式的唯一作者。                                                                        |
| **部分&nbsp;验证** | 提交已签名，签名已成功验证，但提交的作者：a) 不是提交者，并且 b) 已启用警戒模式。 在这种情况下，提交签名并不保证作者的同意，因此提交只得到部分验证。                          |
| **未验证**        | 适用于以下任何情况：<br>- 提交已签名，但无法验证签名。<br>- 提交未签名，提交者启用了警戒模式。<br>- 提交未签名，作者启用了警戒模式。<br> |
