很快掌握所有可用的 Git 命令可能是一项艰巨的任务。 您可以使用“[Git 备忘清单](https://training.github.com/)”快速参考常用的命令。 “Using Git（使用 Git）”备忘清单有多种语言版本。

此外，您可以查看我们的 [Git 和 GitHub 学习资源](/articles/git-and-github-learning-resources/)页面，其中包含许多指南和视频等资源的链接。
