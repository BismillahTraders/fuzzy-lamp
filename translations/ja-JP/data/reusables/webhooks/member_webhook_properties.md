| キー       | 種類       | 説明                                                |
| -------- | -------- | ------------------------------------------------- |
| `action` | `string` | 実行されたアクション. 次のいずれかになります。<ul><li> `added` - ユーザがリポジトリへの招待を受け入れた。</li><li>`removed` - ユーザがリポジトリのコラボレータとして削除された。</li><li>`edited` - ユーザのコラボレータ権限が変更された。 </li></ul> |
