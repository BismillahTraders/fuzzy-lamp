| キー      | 説明                                                                     | 必須 | 種類   | デフォルト                                           | オプション                                           |
| ------- | ---------------------------------------------------------------------- | -- | ---- | ----------------------------------------------- | ----------------------------------------------- |
| `label` | オプションの識別子で、フォームに表示されます。 太字や斜体のテキスト書式、及びハイパーリンクについてMarkdownがサポートされています。 | 必須 | 文字列型 | {% octicon "dash" aria-label="The dash icon" %} | {% octicon "dash" aria-label="The dash icon" %} |
